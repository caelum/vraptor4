--- title: Integrando VRaptor com Tagria --- # Integrando VRaptor com Tagria ## por Jonatan Lemes Tagria é uma suite open source de componentes que te ajuda a construir lindas view em minutos. Tagria integra perfeitamente com qualquer framework MVC baseado em ação incluindo VRaptor Você pode encontrar mais informações visitando o projeto no github Tagria. O primeiro passo é incluir a dependência do Tagria no seu projeto maven como descrito abaixo. ~~~ #!xml com.jslsolucoes tagria-lib 1.0.2 ~~~ E como último passo é necessário declarar a `Servlet` que irá responder pelas requisições feita pelos componentes. No exemplo abaixo, qualquer request para o caminho `/tagria/*` será enviada para os componentes Tagria. ~~~ #!xml tagria com.jslsolucoes.tagria.lib.servlet.Tagria tagria /tagria/* ~~~ E agora nós temos Tagria ouvindo as requisições para construir os componentes. Por default o VRaptor utiliza páginas JSPs. Basta criar uma jsp e incluir as diretivas na sua view. ~~~ #!xml <%@ taglib uri="http://java.sun.com/jsp/jstl/core" prefix="c" %> <%@ taglib uri="http://java.sun.com/jsp/jstl/functions" prefix="fn" %> <%@ taglib uri="http://java.sun.com/jsp/jstl/fmt" prefix="fmt" %> <%@ taglib uri="http://tagrialib.jslsolucoes.com/tags/html" prefix="html"%> <%@ taglib uri="http://tagrialib.jslsolucoes.com/tags/ajax" prefix="ajax"%> <%@ taglib uri="http://tagrialib.jslsolucoes.com/tags/x" prefix="x"%> Minha primeira view ~~~ Você pode encontrar mais documentação sobre todos componentes e exemplos visitando a documentação. Aproveite.
